## 1. 개요: 
**목적:**   피보나치 수열 계산 시 알고리즘의 복잡도에 따른 효율성 차이 확인.

## 3. 분석 및 결론

### 알고리즘 복잡도 비교
1. **재귀적 방법 (Recursive):**
   - 시간 복잡도: **O(2^N)**
   - 원인: 동일한 하위 문제를 중복 계산함에 따라 호출 트리가 지수적으로 증가합니다.
   

2. **순환적 방법 (Iterative):**
   - 시간 복잡도: **O(N)**
   - 원인: 이전 계산값을 메모리에 저장하고 반복문을 한 번만 수행하므로 매우 효율적입니다.
